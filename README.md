# Built with AI

Custom software solutions designed specifically for your business, powered by artificial intelligence.

## Hosting

This site is hosted on GitHub Pages. To deploy:

1. Push this repository to GitHub
2. Go to Settings > Pages
3. Set Source to "Deploy from a branch"
4. Select the main branch and / (root) folder
5. Click Save

The site will be available at https://yourusername.github.io/built-with-ai/
